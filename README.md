# 1 Objetivo do Projeto
Este projeto tem como objetivo analisar dados de vendas para identificar oportunidades de aumento de faturamento, comportamento de produtos e desempenho das categorias. Portanto, gerei insights estratégicos para apoiar a tomada de decisão.

# 2 Tecnologias Utilizadas
- Python
- Pandas
- Power BI
- Excel
- VSCode
- Git Hub

# 3 Processo ETl
## Extração
- Importação dos dados em Excel.

## Transformação
- Tratamento de valores nulos.
- Remoção de duplicados.
- Correção de tipos de dados.
- Validação dos dados.

## Carga
- Exportação do arquivo xlsx tratada para utilização no Power BI

# 4 Análise Expoloratória(EDA)
- Qual faturamento total?
- Quais produtos mais vendem?
- Quais categorias mais geram receita?
- Qual a participação percentual das principais categoria no faturamento total?
- Quais produtos possuem menor desempenho?
- Qual desempenho de cada marca?

# 5 KPIs Criados
- Faturamento Total
- Ticket Médio
- Quantidade de Produtos Vendidos
- Categorias com maior Receita
- Receita de cada Marca
